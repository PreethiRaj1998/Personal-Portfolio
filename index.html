My Projects

Project 1: To-Do List

A simple yet powerful To-Do List application built using HTML, CSS, and JavaScript.
This app allows users to add, save, and delete tasks, ensuring they stay organized. It also utilizes local storage to preserve tasks even after refreshing the page.
Features:
✅ Add Tasks – Easily add new tasks to the list.
✅ Save Tasks – Automatically saves tasks in local storage, so they don’t disappear after a page reload.
✅ Delete Tasks – Remove completed or unnecessary tasks with a single click.
✅ User-Friendly Interface – A clean and responsive design for easy task
✅ Persistent Storage – Uses local storage to keep tasks saved even after closing or refreshing the browser.
Tech Stack: Frontend: HTML, CSS, JavaScript Storage: Local Storage for data persistence How It Works: Add a task by typing in the input field and clicking the "Add" button. View your saved tasks, which remain even after refreshing the page. Delete tasks individually when completed. This project helped me strengthen my JavaScript skills, understand DOM manipulation, and learn how to work with local storage efficiently.
